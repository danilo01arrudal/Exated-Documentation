# TERRAFORM

Terraform é uma ferramenta de infraestrutura como código que permite construir, alterar e versionar infraestrutura com segurança e eficiência. 
Isso inclui componentes de baixo nível, como instâncias de computação, armazenamento e rede; e componentes de alto nível, como entradas DNS e recursos SaaS.

## Entregue infraestrutura como código

  * Adote
    Componha a infraestrutura como código em um arquivo Terraform usando o HCL para provisionar recursos de qualquer provedor de infraestrutura.
  * Construir
    Fluxos de trabalho de automação de infraestrutura para compor, colaborar, reutilizar e provisionar infraestrutura como código em operações de TI e equipes de desenvolvedores.
  * Padronizar
    Estabeleça guarda-corpos para segurança, conformidade e gerenciamento de custos por meio de controles de acesso baseados em funções, aplicação de políticas e auditoria.
  * Escala
    Estenda a automação do fluxo de trabalho para todas as equipes da organização com infraestrutura de autoatendimento como código e integre-se com VCS, ITSM e CI/CD.
